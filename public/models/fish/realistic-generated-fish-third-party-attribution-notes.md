# Optional real CC BY model candidates to download separately

I did not package third-party Sketchfab files in this ZIP. To use a CC BY model, download it yourself from the source page while logged in, save the license proof, and add attribution on the website.

Promising candidates found during research:

- Flounder by brysonchannon on Sketchfab — listed as Creative Commons Attribution.
- Cownose Ray by kenchoo on Sketchfab — listed as Creative Commons Attribution and based on Nicholas DaRocha's cownose ray.
- Stingray/Common Stingray models on Sketchfab — some are CC BY, some are NonCommercial. Verify each page.

CC BY 4.0 generally permits commercial use and adaptation, but requires attribution, license link, and indication of changes.

Use an attribution format like:

> "Model Title" by Creator Name, source link, licensed under CC BY 4.0. Modified/converted to GLB for web display.

Do not use CC BY-NC / NonCommercial assets for the charter website.
